"# MERN-week-8" 
User Management System

A simple full-stack web app to manage users. You can add, view, search, and delete users.

Features
Add new user (Name, Email, Mobile, DOB)
View all users
Search user by name
Delete user
Toast notifications for success/error
Tech Stack

Frontend:

React
Tailwind CSS

Backend:

Node.js
Express
MongoDB
API Endpoints
POST /user-api/user → Add user
GET /user-api/users → Get all users
DELETE /user-api/users/:id → Delete user
